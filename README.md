도시 환경 데이터(오염, 날씨, 녹지, 교통, 인구 등)를 활용해 노원구의 공기질을 분석하고 예측하는 머신러닝 프로젝트

## 📂 주요 파일

| ----------------------------------- | --------------------- |

| `ML2_Classification.ipynb`          | 분류 모델 적용 및 비교         |
| `Final_Presentation.pdf`            | 최종 발표 자료              |

---

## 📊 상세 설명

1. **데이터 수집 및 전처리**

   * 환경 관련 다양한 지표(오염물질, 기상 데이터, 녹지율, 교통량, 인구 통계 등)를 통합
   * 불필요한 컬럼 제거 및 데이터 타입 오류 수정
   * 결측치 처리, 표준화, 이상치 탐지 및 파생 컬럼 생성

2. **모델링 및 성능 비교**

   * 여러 회귀 모델(선형회귀, 랜덤포레스트, XGBoost 등)을 적용해 예측 성능 비교
   * 분류 모델(로지스틱 회귀, 결정트리, SVM 등)로 범주형 공기질 등급 분류
   * 하이퍼파라미터 튜닝과 교차 검증으로 모델 성능 최적화

3. **결과 및 시각화**

   * 모델별 주요 성능 지표(MAE, RMSE, Accuracy 등) 분석
   * 주요 변수 중요도 및 상관관계 시각화
